# 管理员命令

以下是管理员可以使用的所有命令：

## 基本命令

| 命令 | 说明 |
|------|------|
| `/start` | 启动机器人，首次使用可注册为管理员 |
| `/help` | 显示帮助信息（管理员看到完整命令列表） |
| `/cancel` | 取消当前操作，移除底部键盘 |
| `/id` | 查看当前聊天的 Chat ID |

## 管理命令

| 命令 | 说明 |
|------|------|
| `/settings` | 打开交互式设置面板，管理所有配置 |
| `/menu` | 显示配置的菜单内联按钮 |
| `/broadcast <消息>` | 群发消息给所有注册用户（自动附带广播按钮） |
| `/stats` | 查看机器人统计信息（用户数、状态、当前时间） |
| `/users` | 查看已注册用户列表 |
| `/reply <用户ID> <消息>` | 回复指定用户（通过用户 ID） |

## 使用方法

### `/settings` 设置面板

通过 `/settings` 进入交互式设置菜单，可以管理：

- ✏️ 自动回复文字 + ⏱️ 自动删除时间
- 👋 欢迎消息内容
- 🔑 关键词管理（添加/编辑/删除/清空 + 按钮管理）
- ⌨️ 底部快捷键盘管理
- 📢 广播通知（按钮管理）
- 📋 查看当前配置
- 🔄 重载配置

全部通过对话式菜单完成，无需手动编辑配置文件。详见 [设置面板](./features/settings-panel)。

### `/broadcast` 广播

```
/broadcast 大家好，系统将于今晚 23:00 维护。
```

- 直接在命令后输入广播内容即可
- 广播消息会自动附带 `broadcast_buttons` 中配置的按钮
- 完成后会报告成功和失败的统计

详见 [广播消息](./features/broadcast)。

### `/reply` 回复指定用户

```
/reply 123456789 你好，你的问题已处理。
```

- 第一个参数为用户 ID（可通过 `/users` 查看）
- 后面为回复内容
- 适用于需要主动联系某用户的场景

::: tip 推荐使用直接回复
更常用的回复方式是：**直接回复**机器人转发给你的用户消息，机器人会自动识别对应用户，无需输入 ID。详见 [双向通讯](./features/two-way-messaging)。
:::

### `/users` 查看用户列表

显示所有已注册用户，包含：

- 用户名、姓名
- 用户 ID（可用于 `/reply`）
- 加入时间
- 最后活跃时间

### `/stats` 查看统计

```
📊 统计信息

👥 总用户数：15
🤖 机器人状态：运行中
🕐 当前时间：2026-07-18 12:34:56
```

### `/id` 查看聊天 ID

显示当前聊天的 Chat ID，主要用于调试和开发。

### `/menu` 显示菜单

显示 `inline_buttons` 中配置的内联按钮。管理员额外看到功能按钮（如设置、统计等）。

### `/cancel` 取消操作

- 取消当前正在进行的操作（如设置输入、关键字添加等）
- 移除底部快捷键盘
- 发送 `/menu` 可恢复菜单

## 非管理员命令

普通用户可用的命令：

| 命令 | 说明 |
|------|------|
| `/start` | 启动机器人，显示欢迎消息 |
| `/help` | 显示简单提示（「您可以直接发送消息与我们对话」） |
| `/menu` | 显示菜单内联按钮（仅自定义按钮，无管理功能） |
| `/id` | 查看自己的 Chat ID |

::: tip 普通用户使用
普通用户主要功能是发送消息给管理员，无需使用复杂命令。发送任意消息即可触发转发。
:::
