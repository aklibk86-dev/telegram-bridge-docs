# 管理员命令

以下是管理员可以使用的所有命令：

## 基本命令

| 命令 | 说明 |
|------|------|
| `/start` | 启动机器人，首次使用可注册为管理员 |
| `/help` | 显示帮助信息 |
| `/cancel` | 取消当前操作 |

## 管理命令

| 命令 | 说明 |
|------|------|
| `/settings` | 打开交互式设置面板，管理所有配置 |
| `/menu` | 显示配置的菜单内联按钮 |
| `/broadcast` | 群发消息给所有注册用户（支持附带按钮） |
| `/stats` | 查看机器人统计信息 |
| `/users` | 查看已注册用户列表 |
| `/reply` | 回复指定用户（通过用户 ID） |
| `/id` | 查看当前聊天 ID |

## 使用方法

### `/settings` 设置面板

通过 `/settings` 进入交互式设置菜单，可以管理：

- ✏️ 自动回复文本
- ⏱ 自动回复删除时间
- 👋 欢迎消息内容
- 📢 广播消息内容
- 🔑 关键字管理（添加/编辑/删除/清空）
- 🔘 按钮管理（按作用域：菜单/自动回复/欢迎/广播/关键字）
- ⌨️ 底部快捷键盘管理

全部通过对话式菜单完成，无需手动编辑配置文件。

### `/broadcast` 广播

```
/broadcast 大家好，系统将于今晚 23:00 维护。
```

如需附带按钮，可在发送后由机器人引导配置。

### `/reply` 回复指定用户

```
/reply 12345678 你好，你的问题已处理。
```

第一个参数为用户 ID（可通过 `/users` 查看），后面为回复内容。
