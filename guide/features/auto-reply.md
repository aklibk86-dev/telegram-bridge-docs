# 自动回复

TeleBridge 支持灵活的自动回复机制，包括关键字匹配和默认回复。

## 关键字自动回复

在 `config.json` 或通过 `/settings` 面板配置关键字：

```json
{
  "keywords": {
    "价格": {
      "reply": "我们的基础套餐价格为 $99/月",
      "buttons": [{"text": "查看套餐", "url": "https://example.com/pricing"}]
    },
    "工作时间": {
      "reply": "我们的工作时间为周一至周五 9:00-18:00",
      "buttons": []
    }
  }
}
```

### 匹配规则

- 用户消息中**包含**关键字即触发
- 区分大小写？不区分（由 Telegram 客户端决定）
- 匹配后发送配置的回复内容，**不再**转发给管理员

## 默认自动回复

当用户消息不匹配任何关键字时，发送 `auto_reply` 字段配置的默认回复文本。

### 自动删除计时器

设置 `auto_reply_delete_time`（秒），自动回复消息会在指定时间后自动删除：

- 设置为 `10`：10 秒后删除
- 设置为 `0`：不删除（默认）

适用于临时通知场景，减少聊天界面干扰。

## 完整处理流程

```
用户消息
  │
  ├─► 匹配关键字？──是──► 发送关键字回复（带可选按钮）
  │                            │
  │                            ▼
  │                         结束（不转发）
  │
  └─► 无匹配
       │
       ▼
  发送默认自动回复
       │
       ▼
  消息转发给管理员
```
