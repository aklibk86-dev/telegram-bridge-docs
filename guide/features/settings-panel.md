# 交互式设置面板

TeleBridge 最强大的特性之一：**全部配置通过 Telegram 聊天完成**，无需手动编辑 `config.json`。

## 进入设置

向机器人发送 `/settings` 即可进入设置面板。

::: tip 管理员专属
`/settings` 仅管理员可用。非管理员发送时会收到「此命令仅限管理员使用」提示。
:::

## 主菜单

设置面板主菜单包含以下选项：

| 按钮 | 功能 |
|------|------|
| ✏️ 自动回复 | 编辑自动回复文字、设置自动删除时间、管理自动回复按钮 |
| 👋 欢迎消息 | 编辑欢迎消息文字、管理欢迎消息按钮 |
| 🔑 关键词管理 | 添加/编辑/删除/清空关键字 |
| ⌨️ 底部键盘 | 管理管理员底部快捷回复键盘 |
| 📢 广播通知 | 管理广播消息按钮 |
| 📋 查看当前配置 | 查看当前所有配置 |
| 🔄 重载配置 | 从 `config.json` 重新加载配置 |
| ❌ 关闭设置 | 退出设置面板 |

::: tip 配置实时生效
通过设置面板修改的配置会**实时写入** `config.json` 并立即生效，无需重启机器人。
:::

## 各功能详解

### ✏️ 自动回复

进入后可管理：

- **✏️ 编辑自动回复文字**：设置用户消息不匹配关键字时的默认回复
- **⏱️ 设置自动删除时间**：设置自动回复消息的自动删除时间（秒），`0` 表示不删除
- **📋 查看按钮 / ➕ 添加 URL 按钮 / ➖ 删除按钮 / 🗑️ 清空按钮**：管理自动回复消息附带的内联按钮

### 👋 欢迎消息

进入后可管理：

- **✏️ 编辑欢迎消息文字**：设置用户首次 `/start` 时显示的欢迎文本
- **按钮管理**：管理欢迎消息附带的内联按钮（查看/添加/删除/清空）

### 🔑 关键词管理

进入后可管理关键字自动回复：

| 按钮 | 功能 |
|------|------|
| 📋 查看关键词 | 列出所有已配置的关键字 |
| ➕ 添加关键词 | 输入触发词 → 输入回复内容 → 可选添加按钮 |
| 🔧 管理关键词 | 选择已有关键字，编辑回复文字或管理按钮 |
| ➖ 删除关键词 | 选择要删除的关键字 |
| 🗑️ 清空关键词 | 删除所有已配置的关键字 |

**管理单个关键字**时可以：

- ✏️ 编辑回复文字
- 📋 查看按钮（显示当前按钮数量）
- ➕ 添加 URL 按钮
- ➖ 删除按钮
- 🗑️ 清空按钮

### ⌨️ 底部键盘

管理管理员底部的自定义快捷回复键盘布局：

| 按钮 | 功能 |
|------|------|
| 📋 查看底部键盘 | 查看当前键盘布局 |
| ➕ 添加按钮行 | 添加一行按钮（输入按钮文本，多个用空格分隔） |
| 🗑️ 清空键盘 | 清空所有自定义按钮 |
| 🔄 恢复默认 | 恢复默认键盘布局 |

默认键盘布局：

```
⚙️ 设置    📊 统计    👥 用户
📢 广播    ❓ 帮助    🆔 我的ID
```

### 📢 广播通知

进入后可管理：

- **✏️ 编辑广播文字**：编辑广播消息的默认文字模板
- **按钮管理**：管理广播消息附带的内联按钮（查看/添加/删除/清空）

::: tip 广播按钮
广播按钮需预先配置。每次执行 `/broadcast` 命令时，消息会自动附带这些按钮。详见 [广播消息](./broadcast)。
:::

### 📋 查看当前配置

显示当前所有配置的摘要，包括：

- 🤖 Bot Token（部分隐藏）
- 👤 管理员用户名
- 🌐 代理设置
- ✏️ 自动回复文字
- ⏱️ 自动删除时间
- 👋 欢迎消息
- 🔑 关键字数量
- 🔘 各作用域按钮数量

### 🔄 重载配置

从 `config.json` 文件重新加载配置到内存。

适用场景：手动编辑了 `config.json` 文件后，不想重启机器人，可通过此选项立即应用更改。

## 退出设置

以下方式均可退出设置面板：

- 点击主菜单的「❌ 关闭设置」
- 发送 `/cancel` 命令
- 在任何输入步骤中发送「取消」

退出后，机器人恢复正常消息处理模式。

## 常见问题

### 设置时输错了怎么办？

在任何输入步骤中，发送「取消」可取消当前操作并返回上一级菜单，不会保存错误内容。

### 设置会立即生效吗？

是的，通过设置面板修改的所有配置会立即写入 `config.json` 并生效，无需重启。

### 可以手动编辑 `config.json` 吗？

可以，但修改后需：

1. 重启机器人，或
2. 在 `/settings` 中点击「🔄 重载配置」

否则机器人内存中的配置不会更新。
