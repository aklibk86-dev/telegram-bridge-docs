# 交互式设置面板

TeleBridge 最强大的特性之一：**全部配置通过 Telegram 聊天完成**，无需手动编辑 `config.json`。

## 进入设置

向机器人发送 `/settings` 即可进入设置面板。

## 设置菜单

设置面板包含以下主要选项：

### 1. ✏️ 自动回复

设置默认自动回复的文本内容。

### 2. ⏱ 删除时间

设置自动回复消息的自动删除时间（秒），`0` 表示不删除。

### 3. 👋 欢迎消息

设置用户首次 `/start` 时显示的欢迎文本。

### 4. 📢 广播消息

设置默认广播消息的内容（带按钮）。

### 5. 🔑 关键字管理

管理关键字自动回复：
- **添加关键字**：输入触发词 → 输入回复内容 → 可选配置按钮
- **编辑关键字**：选择已有关键字 → 修改回复内容或按钮
- **删除关键字**：选择要删除的关键字
- **清空关键字**：删除所有已配置的关键字

### 6. 🔘 按钮管理

选择一个作用域后进行管理：
- **菜单按钮**：`/menu` 命令显示的按钮
- **自动回复按钮**：自动回复消息附带的按钮
- **欢迎按钮**：欢迎消息附带的按钮
- **广播按钮**：广播消息附带的按钮
- **关键字按钮**：特定关键字的独立按钮

每个作用域支持：添加 / 查看 / 删除 / 清空操作。

### 7. ⌨️ 快捷键盘

管理管理员底部的自定义快捷回复键盘布局。

## 退出设置

发送 `/cancel` 或在任何步骤选择「取消」即可退出设置。
