# 媒体转发

TeleBridge 完整支持各种媒体类型的转发，用户发送的媒体消息会转发给管理员。

## 支持的媒体类型

| 类型 | 说明 |
|------|------|
| 📷 图片 | 单张图片 |
| 🎥 视频 | 视频文件 |
| 📄 文档 | PDF、Word、Excel 等 |
| 🎵 音频 | 音乐文件 |
| 🎤 语音 | Telegram 语音消息 |
| 🎞 动画 | GIF 动画 |
| 🏷 贴纸 | Telegram 贴纸 |

## 管理员回复媒体

管理员可以直接回复转发的媒体消息，机器人会将回复内容发回给用户。

## 注意事项

- 媒体消息**不触发**关键字自动回复（只转发给管理员）
- 媒体大小受 Telegram API 限制（最大 50MB）
- 转发时保留原始消息的媒体信息
